### HTA 1
<img width="1850" height="650" alt="image" src="https://github.com/user-attachments/assets/e6ed9427-c5d3-459f-b6e9-8f538c4ff6f6" />

Para realizar o objetivo de uma transação Pix, o usuário deve seguir uma sequência linear de ações. O processo começa ao acessar a opção Pix no sistema. Em seguida, é necessário escolher o destinatário através de uma chave Pix, onde o usuário deve selecionar uma opção entre e-mail, número de telefone, CPF, chave aleatória ou um contato já salvo.
Após definir o destinatário, o usuário define o valor da transação. A etapa seguinte é a verificação dos dados de quem vai receber; neste ponto, o usuário deve decidir entre confirmar para continuar ou cancelar/corrigir as informações se houver erro. Uma vez confirmados os dados, é realizada uma autenticação dupla por segurança e, por fim, a operação é concluída na etapa de finalizar transferência.
