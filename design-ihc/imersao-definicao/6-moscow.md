# Moscow - Funcionalidades Gerais 

### Lista bruta do MOSCOW

- Integrações com bancos digitais/carteira digitais nacionais.  

- Cartão de Débito Exclusivo do APP.  

- Integração com um ChatBot de IA.  

- Tutorial Dinâmico para o usuário.  

- Implementação para outros tipos de acessibilidades (Possíveis deficiências)  

- Sistema de reconhecimento facial para autenticação segura e rápida no APP.   

- Sistema de biometria para autenticação segura e rápida no APP.   

- Monitoramento em tempo real de transações suspeitas  

- Alertas imediatos de falhas de segurança 

- Opção de bloqueio instantâneo do cartão.  

- Outros meios de transferência bancária.  

- Lista de chaves pix favoritadas. 

- Personalização (temas, cores e modo escuro/claro). 

- Confirmação clara de cada transação. 

- Histórico de todas as transações realizadas no APP por data e horário 

- Interface minimalista e intuitiva. 

- Educação financeira detalhada dentro do app. 

- Central de ajuda com linguagem simples e vídeos explicativos curtos. 

- Sistema de reconhecimento de padrões de uso, que alerta se o usuário tentar algo fora do comum (prevenção de golpes). 

- Compartilhamento de comprovante. 

- O sistema terá funcionalidades como empréstimos ou pagamento de boletos. 

- O sistema terá alta densidade de informações ou ofertas de crédito/investimentos na interface. 

- Relatórios mensais simplificados com resumo das transações Pix.
